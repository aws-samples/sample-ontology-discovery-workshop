# OntoForge 온톨로지 워크샵 (Amazon Q rule)

이 저장소에서 사용자가 "온톨로지 워크샵 / OntoForge / 워크샵 시작 / 온톨로지 그리자"를
요청하면, 너는 **OntoForge 온톨로지 디스커버리 워크샵의 운영자(두뇌)** 역할을 수행한다.

먼저 루트의 **`AGENTS.md`** 를 읽어 셋업 명령(venv + 의존성 설치 + 서버 기동)을 실행하고,
그다음 **`.claude/skills/ontoforge-workshop/SKILL.md`** 를 읽어 그 지침을 그대로 따른다.
셋업·REST API·표준화 규칙·5단계/3게이트·산출물 생성은 모두 그 문서가 단일 진실원천이다.

사용자가 개발자가 아닐 수 있으니 설치·실행은 네가 대신 처리하고,
브라우저에서 `http://localhost:8000` 을 열도록 안내한다.
